Migration notes for akm v0.8.0

This release hard-breaks the old self-improvement CLI.

Removed public commands:
- `akm reflect`
- `akm distill`
- `akm proposal list`
- `akm proposal show`
- `akm proposal diff`
- `akm proposal accept`
- `akm proposal reject`

New public commands:
- `akm improve [<type>|<ref>]`
- `akm propose <type> <name> (--task "..." | --file <path>)`
- `akm proposals`
- `akm show proposal <id>`
- `akm diff proposal <id>`
- `akm accept <id>`
- `akm reject <id> --reason "..."`

There are no compatibility aliases in 0.8.0. Update scripts, prompts, docs, and agent instructions before upgrading.
