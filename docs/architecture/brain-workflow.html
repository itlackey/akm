Neuroscience · Cognitive Architecture

How the Brain Turns Perception
into Knowledge

A high-level map of the neural workflow — from raw sensory signal to consolidated long-term memory, pattern abstraction, and functional understanding.

Stage 01

Sensory Perception

Raw signals arrive from all five senses simultaneously — light, sound, pressure, chemical gradients. Each sense has dedicated receptor cells that transduce physical stimuli into electrical nerve impulses.

↳ Eyes · Ears · Skin · Nose · Tongue → Cranial nerves
Stage 01b

Sensory Buffer

Signals are held in ultra-short-term sensory registers (iconic / echoic memory) for ~250ms–2s. This is a pre-conscious holding area — most of this stream is immediately discarded.

↳ Primary sensory cortices (V1, A1, S1)
↓ Thalamic relay · Attentional filtering ↓
Stage 02

Attention Gate & Working Memory

The thalamus acts as a relay and gating station. The prefrontal cortex selects what gets elevated to conscious focus. Selected content enters working memory — the brain's "RAM" — capable of holding ~4 items at once for active manipulation.

↳ Thalamus · Prefrontal Cortex · Anterior Cingulate
↓ Emotional salience tagging runs in parallel ↓
Stage 03a · Hippocampal Encoding

Short-Term Consolidation

The hippocampus binds disparate cortical signals into a single "episode" — binding who, what, where, and when into a coherent memory trace. This binding is fragile and requires rehearsal or emotional weight to persist.

↳ Hippocampus · Entorhinal Cortex
Stage 03b · Emotional Tagging

Amygdala Salience Marking

The amygdala assigns an emotional weight (fear, reward, novelty) to each experience in real time. High-salience events trigger norepinephrine & cortisol release — chemically marking the memory as "important, keep this."

↳ Amygdala · Locus Coeruleus · VTA (dopamine)
↓ Offline processing during sleep ↓
Stage 04 · Offline Consolidation

Sleep — The Memory Compiler

Sleep is not passive. It is when the brain performs its most critical integration work — replaying, compressing, and transferring the day's experiences into stable long-term storage. This happens across two distinct sleep architectures:

NREM / Slow-Wave Sleep

Hippocampal "sharp-wave ripples" replay compressed sequences of the day's episodes. These fire in sync with cortical slow oscillations, transferring episodic traces to distributed neocortical storage. Declarative memory consolidation happens here.

REM Sleep · Dreaming

The prefrontal cortex largely disengages. The brain free-associates across loosely related memory traces — testing novel connections between disparate concepts. Emotional tone is stripped from memories (amygdala dampened by serotonin withdrawal). Dreams may be the subjective experience of this associative cross-indexing.

Glymphatic Clearance

Cerebrospinal fluid flushes the interstitial space of metabolic waste (including amyloid-β). Sleep deprivation disrupts this — accumulation impairs both encoding and retrieval the next day.

↳ Hippocampus → Neocortex transfer · Thalamo-cortical spindles · Brainstem (REM atonia)
↓ Synaptic weight changes via LTP/LTD ↓
Stage 05a · Explicit Memory

Declarative Memory Storage

Episodic memories (events) and semantic memories (facts, concepts) are stored as distributed patterns of synaptic weights across association cortices. There is no single "storage location" — a memory is a reconstruction each time it is recalled.

↳ Temporal · Parietal · Frontal association cortices
Stage 05b · Implicit Memory

Procedural & Implicit Storage

Skills, habits, and conditioned responses bypass the hippocampus entirely. Repeated motor sequences are "compiled" into the basal ganglia and cerebellum — becoming automatic, fast, and largely inaccessible to verbal introspection.

↳ Basal Ganglia (habits) · Cerebellum (motor) · Striatum
↓ Cross-episodic abstraction · Schema formation ↓
Stage 06

Pattern Recognition & Schema Formation

As memories accumulate and overlap, the brain extracts statistical regularities. Individual episodes are compressed into abstract schemas — generalized templates of "how the world works." New information is rapidly assimilated if it fits an existing schema; schemas update slowly under repeated contradiction.

↳ Medial Prefrontal Cortex · Neocortical hierarchies · Default Mode Network
↓ Retrieval · Application · Prediction ↓
Stage 07 · Output Layer

Functional Knowledge & Predictive Modeling

The end state is not passive storage — it is an active predictive model of the world. The brain constantly generates predictions about incoming sensory data and only updates when those predictions are violated (prediction error signals). Knowledge manifests as:

Conceptual Understanding

Flexible, context-independent knowledge that can be reasoned about, communicated, and transferred to novel domains.

Skilled Performance

Automatized procedural sequences that execute rapidly and accurately below the threshold of conscious attention.

Predictive Intuition

Pattern-matched fast heuristics — "gut feeling" — that compress years of accumulated schema into near-instant recognition and judgment.

↳ Prefrontal Cortex (executive function) · Default Mode Network (self-modeling) · Dopamine system (prediction error / reward)
⟳   This is a closed loop, not a pipeline. Every act of retrieval slightly rewrites the memory (reconsolidation). New perception is constantly filtered through existing schemas. Prediction errors drive learning. The "output" of Stage 07 feeds directly back into Stage 02 as a top-down attentional prior — we literally perceive through the lens of what we already know.
